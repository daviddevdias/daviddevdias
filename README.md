## 🧠 Sobre Mim

> **Opa, e aí! Sou o David, de Esteio (RS), e estudo Análise e Desenvolvimento de Sistemas.** Hoje meu foco é Front-End, mas a real é que eu curto mesmo é resolver problemas, não importa a stack.

Gosto daquele processo de pegar uma ideia, desenhar no Figma e ver o negócio criando vida na tela com HTML, CSS e JS. Mas não fico só no visual não: vira e mexe eu tô no back-end ou fazendo uns scripts em Python e Node.js pra automatizar o que dá.

Meu objetivo a longo prazo é ser Full Stack. Tô sempre fuçando tecnologias novas e agora procuro uma oportunidade na área pra colocar a mão na massa em projetos reais, quebrar a cabeça com problemas de verdade e crescer junto com a equipe.
