## 🔥 Featured Project

### 🤖 Jarvis Assistant

💡 Sistema inteligente de automação desenvolvido em Python com arquitetura modular.

---

### ⚙️ O que ele faz

- 🧠 Interpreta comandos personalizados  
- 📂 Gerencia arquivos (CRUD + backup + organização)  
- 📊 Monitora sistema (CPU, disco)  
- 🌐 Integra com APIs externas  
- ⚡ Automatiza tarefas do dia a dia  

---

### 🏗️ Destaques técnicos

- Arquitetura modular e escalável  
- Separação de responsabilidades (controllers, modules, memory)  
- Uso de `psutil`, `pathlib`, `pyautogui`  
- Sistema de logs e tratamento de erros  
- Estrutura pronta para evolução (API / web)  

---

### 🚀 Objetivo

Evoluir o projeto para um assistente completo com API e interface web.
